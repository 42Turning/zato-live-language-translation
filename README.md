# Z.A.T.O. 实时双语切换补丁

## 简介

为视觉小说 **Z.A.T.O. // I Love the World and Everything in It** 添加**按 T 键实时切换语言**功能。主要用于中文玩家快速获得中文翻译。

## 功能

- **按 T 键**：当前对话瞬间切换为中文
- **前进后自动还原**：点击/回车进入下一句时，自动切回英文
- **设置界面集成**：打开游戏设置（Esc → Settings），左下角有选项：
  - **自动前进 / 手动前进**：还原英文后是否自动进入下一句
  - **跳过动画 / 保留动画**：还原英文时是否跳过文字动画

## 安装

将 `toggle_language.rpyc` 复制到游戏目录的 `game/` 文件夹下：

```
Z.A.T.O.  I Love the World and Everything In It/
├── game/
│   ├── toggle_language.rpyc    ← 放这里
│   ├── scripts.rpa
│   └── ...
├── ZATO.exe
└── ...
```

**注意**：此补丁依赖已有的中文汉化补丁[[人工\双语]《Z.A.T.O.》汉化补丁](https://steamcommunity.com/sharedfiles/filedetails/?id=3651682454)，确保中文翻译文件存在。
[【【汉化】《Z.A.T.O.》完整汉化发布：1986年，苏联，一座不存在的小镇，一个女孩的故事】](https://www.bilibili.com/video/BV1AaSTBEEmf) 作者:坚果和高坚果。感谢作者的无私奉提供这么高质量的翻译。

## 使用

建议使用默认字体，可以获得更好的体验

| 操作 | 效果 |
|---|---|
| **按 T** | 英文 → 中文（瞬间切换） |
| **点击/回车** | 还原英文 → 进入下一句 |
| **打开设置** | 调整自动前进/跳过动画选项 |

自动前进和手动前进是翻译这一句之后，点击自动出现下一句，手动是点击后这一句重新出现，方便参考。

跳过动画就是当前想要翻译的这句话重复出现时候，文字是否需要直接出现，不影响之后的阅读。

注意，请保证键盘状态在英文小写输入的状态。而且经过测试如果保证大写输入(在大写锁定的状态)游戏会崩溃，但经过搜索好像是游戏引擎问题(也可能是我的问题)。所以这个补丁暂时放弃兼容大写，之后会完善，感谢理解。
